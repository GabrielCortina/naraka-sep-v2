## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gap in stat cards, medal-to-name gap in rankings |
| sm | 8px | Gap between stat cards on mobile, padding inside ranking rows |
| md | 16px | Padding inside dashboard blocks, gap between block header and content |
| lg | 24px | Gap between dashboard blocks (vertical stack), section padding |
| xl | 32px | Gap between left and right columns on desktop |
| 2xl | 48px | Top padding from nav to first block (mobile) |
| 3xl | 64px | Top padding from nav to first block (desktop) |

Exceptions: Stat card number uses 36px font which is not on the spacing scale but is a typography value. Medal emoji size is 20px (inherits from text size, not a spacing token).

## Typography

| Role | Size | Weight | Line Height | Usage in This Phase |
|------|------|--------|-------------|---------------------|
| Label | 12px (text-xs) | 600 (semibold) | 1.5 | Block headers ("RESUMO GERAL", "TOP SEPARADORES"), stat card labels, "pecas" / "cards" sub-labels in rankings, period filter label |
| Body | 14px (text-sm) | 400 (normal) | 1.5 | Ranking names, separador progress rows, method names in Progressao, "X/Y pecas" counts, countdown text |
| Heading | 20px (text-xl) | 600 (semibold) | 1.2 | Block titles when expanded, period filter selected value |
| Display | 36px (text-4xl) | 600 (semibold) | 1.1 | Stat card large numbers (pecas separadas, pendentes, concluidas, em atraso) |

Font sizes: 4 declared (12px, 14px, 20px, 36px).
Font weights: 2 declared (400 normal, 600 semibold).

Source: Established project patterns from Phase 5 and Phase 8. Display bumped to 36px (from 28px in Phase 8) because stat cards need prominent numbers visible at a glance from across the warehouse floor.

## Interaction States

### Block States

| State | Visual |
|-------|--------|
| Loading | Skeleton placeholders (bg-muted animate-pulse) in each block |
| Loaded (collapsed) | Summary data visible, chevron pointing down |
| Loaded (expanded) | Detail list visible below summary, chevron pointing up |
| Realtime update | Data re-fetched on Supabase subscription event; no visual flash (D-12) |

### Period Filter States

| State | Visual |
|-------|--------|
| Default | "Hoje" selected in dropdown |
| Open | shadcn Select dropdown with 7 options |
| Custom selected | Two date inputs appear below Select (De / Ate) |
| Loading period data | Skeleton rows replace ranking content while fetching |

### Stat Card States

| State | Visual |
|-------|--------|
| Normal | Number in foreground, label in muted-foreground |
| "Em Atraso" with count > 0 | Number in destructive color, card background bg-destructive/5 |
| "Em Atraso" with count = 0 | Normal styling (no red tint when nothing is late) |

### Empty Day State

| State | Visual |
|-------|--------|
| No data (before first upload) | All stat cards show "0", progress blocks show empty message, rankings show empty message |

## Accessibility Notes

| Concern | Solution |
|---------|----------|
| Block expansion | Collapsible trigger has aria-expanded, content has role="region" (shadcn default) |
| Stat card semantics | Use aria-label on each stat card: e.g., "Pecas separadas: 142" |
| Progress bars | ProgressBar includes aria-valuenow, aria-valuemin, aria-valuemax |
| Period filter | shadcn Select handles keyboard navigation and ARIA |
| Medal emojis | Decorative only -- screen readers read position number + name + metric |
| Color contrast | Stat card numbers on white background meet WCAG AA. Amber/blue/green left borders are decorative (label text provides meaning) |
| Realtime updates | Use aria-live="polite" on stat card numbers for screen reader announcements on change |
