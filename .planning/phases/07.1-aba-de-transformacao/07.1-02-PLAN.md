**1. Create `src/features/transformacao/hooks/use-transformacao-data.ts`:**

Custom hook that fetches from `transformacoes` table and assembles `TransformacaoCardData[]`. Pattern follows `use-card-data.ts` but is simpler since transformacoes rows ARE the items (no pedidos+progresso+reservas join).

```typescript
'use client'
import { useState, useEffect, useCallback } from 'react'
import { createClient } from '@/lib/supabase/client'
import type { TransformacaoItem, TransformacaoCardData } from '../types'
```

**fetchTransformacoes function:**
- `supabase.from('transformacoes').select('*')` (use `as any` cast like use-card-data.ts does for the same table)
- Group rows by `card_key + numero_transformacao` into a Map
- For each group, parse `card_key` to extract grupo_envio, tipo, importacao_numero using split('|'):
  ```typescript
  const parts = cardKey.split('|')
  const grupo_envio = parts[0] ?? ''
  const tipo = parts[1] ?? ''
  const importacao_numero = parseInt(parts[2] ?? '0', 10)
  ```
- Compute per-card:
  - `total_pecas`: sum of all items' quantidade
  - `pecas_concluidas`: sum of quantidade for items with status='concluido'
  - `atribuido_a`: from first item with separador_id/separador_nome (all items in a card share assignment)
  - `status`: 'concluido' if all items concluido, 'atribuido' if any item has separador_id, else 'pendente'

**Visibility filtering (D-16):**
- If userRole is 'separador': filter cards where `atribuido_a?.id === userId` OR cards with no assignment (status='pendente'). Wait -- per D-16, "Itens sem atribuicao aparecem so para lider/admin". So separador sees ONLY cards where atribuido_a?.id === userId.
- If userRole is 'admin' or 'lider': see all cards.

**Realtime subscription:**
- Create a SEPARATE channel named `'transformacao-realtime'` (NOT the cards-realtime channel)
- Subscribe to `postgres_changes` on table `transformacoes` for events `*` (INSERT, UPDATE, DELETE)
- On any change, call `fetchTransformacoes()` to refetch all data

Return `{ cards: TransformacaoCardData[], loading: boolean, error: string | null }`.
