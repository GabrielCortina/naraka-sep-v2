**GREEN:** Criar `src/features/fardos/utils/subset-sum.ts` com a funcao `findOptimalCombination(fardos: StockItem[], demanda: number): SubsetResult`.

Algoritmo (DP com programacao dinamica):
1. Se fardos vazio, retornar `{ fardos: [], soma: 0, cobertura: 'nenhuma' }`
2. Calcular `maxTarget = Math.ceil(demanda * 1.20)` (limite superior de 20%)
3. Usar DP para encontrar todas as somas possiveis ate `maxTarget`:
   - Tabela DP: `Map<number, { fardos: StockItem[], count: number }>` onde key=soma, value=melhor combinacao para aquela soma
   - Para cada fardo, iterar somas existentes e adicionar novas
4. Selecionar melhor resultado:
   a. Primeiro, buscar soma exata `== demanda` -- se encontrar, retornar com `cobertura: 'total'`
   b. Buscar somas no range `[demanda, maxTarget]` -- a menor soma nesse range vence. Desempate: menos fardos (D-11)
   c. Se nenhuma soma >= demanda dentro de 20%, buscar a maior soma < demanda -- retornar com `cobertura: 'parcial'`
   d. Se nenhuma soma possivel, retornar `cobertura: 'nenhuma'`

IMPORTANTE: Na etapa 4b, entre somas iguais, preferir combinacao com menos fardos. Na etapa 4a (soma exata), tambem aplicar desempate por menos fardos.

A funcao NAO deve ter side effects -- funcao pura que recebe dados e retorna resultado.
