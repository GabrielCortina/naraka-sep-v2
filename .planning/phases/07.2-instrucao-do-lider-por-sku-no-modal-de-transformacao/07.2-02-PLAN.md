```typescript
  { value: 'PEGAR_NA_LOJA', label: 'Pegar na Loja', color: 'bg-purple-500' },
]

interface InstrucaoPopoverProps {
  transformacaoId: string
  instrucaoAtual: InstrucaoLider | null
}

export function InstrucaoPopover({ transformacaoId, instrucaoAtual }: InstrucaoPopoverProps)
```

Comportamento:
- Trigger: Icone `MessageSquare` (16x16), botao quadrado 7x7 (h-7 w-7), borda cinza, hover escurece
- Se `instrucaoAtual` nao e null, o trigger tem borda colorida correspondente (visual hint)
- Ao clicar numa opcao:
  - `setLoading(true)`
  - `fetch('/api/transformacao/instrucao', { method: 'PATCH', body: JSON.stringify({ transformacao_id: transformacaoId, instrucao: opcao.value }) })`

  - Se erro: `toast.error(data.error || 'Erro ao definir instrucao')`

- Cada opcao renderizada como botao com: bolinha colorida (w-3 h-3 rounded-full) + label texto
- Opcao selecionada tem check mark ou fundo levemente destacado
- PopoverContent: `w-56 p-2`
