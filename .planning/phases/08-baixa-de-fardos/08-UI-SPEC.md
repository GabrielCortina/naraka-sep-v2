## Component Inventory

### 1. BaixaInput (campo de busca)

| Property | Spec |
|----------|------|
| Component | shadcn Input + custom wrapper |
| Width | 80% of viewport on mobile, 50% on desktop (D-02) |
| Height | 56px (h-14) |
| Font size | 18px (text-lg) |
| Placeholder | "Escanear ou digitar codigo IN" |
| Auto-focus | On mount and after every baixa confirmation (D-01, D-13) |
| Trigger | Enter key only -- no onChange debounce (D-01, Pitfall 6) |
| Error state | border-red-500 for 2 seconds, then revert (D-04) |
| Camera button | 40px square icon button to the right with Camera icon from lucide-react (D-03) |
| Layout | Centered horizontally on both mobile and desktop |

### 2. BaixaModal (confirmacao)

| Property | Spec |
|----------|------|
| Component | shadcn Dialog (centered popup per D-05) |
| Border-top | 4px solid in marketplace color (D-06) |
| Max width | 420px on desktop, 90vw on mobile |
| Padding | 24px |
| Sections (top to bottom) | Codigo IN (heading, 20px semibold), SKU (body, 14px), Endereco (body, 14px, with MapPin icon in green), Quantidade ("CONTEM" label 12px semibold + value 28px bold), "Entregar para:" section |
| MapPin icon | 16px, color urgency-ok (green), inline with endereco text (D-05) |
| Quantidade | Label "CONTEM" in 12px semibold uppercase above, number in 28px bold below |
| Animation | Default shadcn Dialog animation (fade + scale) |

### 3. Entregar Para Section (inside modal)

| Property | Spec |
|----------|------|
| Header | "Entregar para:" in 12px semibold, muted-foreground color |
| List | Vertical, each line = separador nome + card_key |
| Line format | "{nome}" in 14px semibold + "{card_key}" in 12px normal muted-foreground |
| Line accent | Left border 3px solid in marketplace color of that card (D-09) |
| Line padding | 8px vertical, 12px left (after border) |
| Not assigned | "Nao atribuido" in 14px italic + card_key in 12px muted (D-11) |
| Scroll | Internal scroll if more than 4 entries (D-12), max-height 200px |

### 4. Modal Action Buttons

| Property | Spec |
|----------|------|
| "Confirmar Baixa" | shadcn Button, bg green (urgency-ok), text white, full width, 48px height, 14px semibold |
| "Cancelar" | shadcn Button variant="outline", muted colors, full width, 44px height, 14px normal |
| Layout | Stacked vertically, Confirmar on top, Cancelar below, 8px gap |
| Loading state | Spinner icon (Loader2 from lucide-react) + disabled + opacity-50 (D-07) |
| Double-click prevention | disabled={isLoading} on both buttons during processing (D-07) |

### 5. BaixadosHoje Section

| Property | Spec |
|----------|------|
| Component | shadcn Collapsible |
| Header | "BAIXADOS HOJE ({count})" in 12px semibold uppercase, with ChevronDown/Up icon |
| Initial state | Collapsed. Expands automatically when count goes from 0 to 1 (D-15) |
| Background | secondary (hsl 0 0% 96.1%) |
| Border radius | 8px (var(--radius)) |
| Position | Below input area, separated by 32px gap |
| Width | Same as input area (80% mobile, 50% desktop) |

### 6. BaixadosHoje List Items

| Property | Spec |
|----------|------|
| Layout | Single row per baixa: codigo IN, SKU, quantidade, separador(es), horario |
| Mobile layout | Two lines: line 1 = "IN-XX | SKU | Qtd", line 2 = "Para: nome | HH:MM" |
| Desktop layout | Single row with columns |
| Font | 14px normal for data, 12px muted-foreground for timestamps |
| Separator | 1px border-bottom between items (border color) |
| Padding | 8px vertical per item |
| Empty state | Not visible -- section stays collapsed when count = 0 |

### 7. CameraScanner (optional overlay)

| Property | Spec |
|----------|------|
| Component | @yudiel/react-qr-scanner wrapped in dynamic import (lazy loaded) |
| Trigger | Camera icon button next to input field (D-03) |
| Layout | Full-screen overlay with dark background (bg-black/80) |
| Viewfinder | Centered square, 280px on mobile |
| Close button | X icon top-right, white, 32px |
| On scan | Extract barcode value, close overlay, populate input, auto-trigger search |
| Unmount | Immediately after successful scan or close (performance per Pitfall 4) |

### 8. Toast Notifications

| State | Style | Copy | Duration |
|-------|-------|------|----------|
| Not found (D-04) | destructive variant (red tint) | "Fardo nao encontrado no trafego" | 3s |
| Already discharged (D-08) | warning (amber tint) | "Fardo {codigo_in} ja teve baixa" | 3s |
| Success (D-13) | success (green tint) | "Baixa confirmada -- {codigo_in}" | 2s |

Source: sonner (already installed), CONTEXT.md D-04/D-08/D-13

## Interaction States

### Input Field States

| State | Visual |
|-------|--------|
| Idle (focused) | Default border, blinking cursor, auto-focused |
| Typing | Text appears in 18px, no search triggered |
| Error flash | border-red-500 for 2 seconds, then returns to default (D-04) |
| Processing | Input disabled while modal is open |

### Modal States

| State | Visual |
|-------|--------|
| Opening | Default shadcn Dialog animation (150ms fade + scale) |
| Loaded | Full content displayed, buttons enabled |
| Confirming | "Confirmar Baixa" shows spinner, both buttons disabled, opacity-50 on Cancelar |
| Closing (success) | Modal closes, toast appears, input field clears and re-focuses |
| Closing (cancel) | Modal closes, input field clears and re-focuses |

### BAIXADOS HOJE States

| State | Visual |
|-------|--------|
| Empty (count=0) | Section collapsed, header shows "BAIXADOS HOJE (0)" |
| First baixa | Section auto-expands with animation (D-15) |
| Populated | List of items, most recent at top, scrollable if needed |

## Accessibility Notes

| Concern | Solution |
|---------|----------|
| Input auto-focus | useEffect focus on mount; re-focus after modal close |
| Modal keyboard | shadcn Dialog handles Escape to close, Tab trap, focus return |
| Toast announcements | sonner handles aria-live regions |
| Camera scanner | Full-screen overlay with visible close button; Escape to dismiss |
| Color contrast | Green button (#22C55E) on white meets WCAG AA for large text (48px button); marketplace colors used only as decorative borders, not for text |
