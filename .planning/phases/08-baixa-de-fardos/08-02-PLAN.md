    - `search(codigoIn: string)`:
      - Set isSearching = true, hasError = false
      - Fetch `GET /api/baixa/buscar?codigo_in=${encodeURIComponent(codigoIn)}`
      - If response.status === 409 (duplicado): toast.warning(`Fardo ${codigoIn} ja teve baixa`) per D-08. Set isSearching = false. Return { clear: true }
      - If response.status === 404: toast.error('Fardo nao encontrado no trafego') per D-04. Set hasError = true. setTimeout(() => setHasError(false), 2000). Set isSearching = false. Return { clear: true }
      - If response.ok: set fardo = data, set modalOpen = true. Set isSearching = false. Return { clear: false } (don't clear input yet)
      - On network error: toast.error('Erro de conexao'). Set isSearching = false. Return { clear: true }
    - `confirm()`:
      - Guard: if !fardo, return
      - Set isConfirming = true
      - Fetch `POST /api/baixa/confirmar` with body `{ codigo_in: fardo.codigo_in, trafego_id: fardo.trafego_id }`
      - If response.ok: toast.success(`Baixa confirmada -- ${fardo.codigo_in}`) per D-13. Add to baixadosHoje (prepend -- build BaixadoItem from current fardo data including entregas). Close modal. Set isConfirming = false. Return { success: true }
      - If response.status === 409: toast.warning(`Fardo ${fardo.codigo_in} ja teve baixa`). Close modal. Set isConfirming = false. Return { success: false }
      - On error: toast.error('Erro ao confirmar baixa'). Set isConfirming = false. Return { success: false }
    - `cancel()`: Set modalOpen = false, fardo = null
    - `loadBaixadosHoje()`:
      - Called on mount via useEffect.
      - Fetch `GET /api/baixa/hoje` to get full BaixadoItem[] with entregas populated (per D-14 "para quem foi entregue").
      - Set baixadosHoje from response data.
      - This endpoint (created in Plan 01) joins baixados -> trafego_fardos -> reservas -> pedidos -> atribuicoes to build the full entregas array for each historical baixado.
    - Realtime subscription for baixados table: listen for INSERT events. On change, call loadBaixadosHoje() to refresh the list (when another fardista does a baixa). Use same pattern as useCardsRealtime but on a 'baixa-realtime' channel listening to baixados table.
    - Export hook returning: `{ fardo, isSearching, isConfirming, modalOpen, hasError, baixadosHoje, search, confirm, cancel }`
