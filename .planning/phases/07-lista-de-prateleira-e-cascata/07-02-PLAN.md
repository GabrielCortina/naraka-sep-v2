    1. Create `app/api/prateleira/cascata/route.ts` -- POST route handler:
       - **Auth** (same pattern as fardos/ne): `createClient()` -> `getUser()` -> `supabaseAdmin.from('users').select('role, nome').eq('id', user.id).single()`. Allowed roles: `['separador', 'admin', 'lider']`
       - **Body**: `{ pedido_ids: string[], sku: string, quantidade_restante: number, tipo: 'parcial' | 'ne', card_key: string }`
       - **Validation**: pedido_ids must be non-empty array of strings, sku non-empty string, quantidade_restante > 0, tipo must be 'parcial' or 'ne', card_key non-empty string
       - **Server-side quantity validation**: fetch pedidos from DB for given pedido_ids, sum their `quantidade`. Verify `quantidade_restante <= totalQuantidade`. Return 400 if exceeds.
       - **Step 1 - Update progresso** for each pedido_id:
         - If tipo='parcial': select-then-insert/update progresso with `status='parcial'`, `quantidade_separada` = (quantidade from body - quantidade_restante) distributed across pedido_ids proportionally. Use same select-then-insert/update pattern as /api/cards/progress.
         - If tipo='ne': select-then-insert/update progresso with `status='nao_encontrado'`, `quantidade_separada=0` for each pedido_id
       - **Step 2 - Build exclusion sets** (same as fardos/ne pattern):
         ```typescript
         const { data: reservasAtivas } = await supabaseAdmin.from('reservas').select('codigo_in').eq('status', 'reservado')
         const reservedSet = new Set((reservasAtivas ?? []).map(r => r.codigo_in))

         const hoje = new Date()
         hoje.setHours(0, 0, 0, 0)
         const { data: naoEncontradosHoje } = await supabaseAdmin.from('fardos_nao_encontrados').select('codigo_in').gte('reportado_em', hoje.toISOString())
         const naoEncontradosSet = new Set((naoEncontradosHoje ?? []).map(r => r.codigo_in))
         ```
       - **Step 3 - Fetch stock**: `const stock = await fetchStock(true)` (force refresh per Pitfall 5)
       - **Step 4 - Run cascade**: `const result = findCascadeBales(stock, sku, quantidade_restante, reservedSet, naoEncontradosSet)`
       - **Step 5 - Process result**:
         - If `result.fardos.length > 0`:
           - For each bale in result.fardos:
             - Insert into `reservas`: `{ codigo_in, sku, quantidade, endereco, status: 'reservado', importacao_numero }` (get importacao_numero from first pedido)
             - Insert into `trafego_fardos` (cast as any for migration 00006 cols): `{ codigo_in, sku, quantidade, endereco, status: 'pendente', reserva_id: newReserva.id, is_cascata: true }`
           - Update progresso for pedido_ids: set status to `'aguardar_fardista'` for the cascaded portion (per D-05)
             - If tipo='parcial': only update the portion that wasn't directly confirmed
             - If tipo='ne': update all pedido_ids to 'aguardar_fardista'
         - If `result.quantidade_transformacao > 0`:
           - Get separador info: query atribuicoes for card_key tipo='separador' with users join
           - Insert into `transformacoes` (cast as any): `{ sku, quantidade: result.quantidade_transformacao, card_key, numero_pedido: (first pedido's numero_pedido), separador_id, separador_nome, status: 'pendente' }`
           - Update progresso for pedido_ids: set status to `'transformacao'` for the transformacao portion (per D-16)
       - **Step 6 - Return**: `{ found_alternative: result.fardos.length > 0, bales: result.fardos.map(f => ({ codigo_in: f.codigo_in, quantidade: f.quantidade })), transformacao: result.quantidade_transformacao > 0, quantidade_transformacao: result.quantidade_transformacao }`
       - **Error handling**: wrap stock fetch in try/catch, return 502 on stock error. Handle unique constraint (23505) on reservas.codigo_in gracefully (return 409 "Fardo ja reservado, tente novamente")
