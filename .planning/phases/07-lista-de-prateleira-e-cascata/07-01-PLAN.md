         ```typescript
         export interface CascadeResult {

         }
         ```
       - Export function `findCascadeBales(stock: StockItem[], sku: string, demanda: number, reservedSet: Set<string>, naoEncontradosSet: Set<string>): CascadeResult`
       - Import `StockItem` from `@/features/fardos/types`
       - Algorithm (per D-06, D-07, D-08):
         a. Filter `disponiveis` = stock items where `item.sku === sku && !reservedSet.has(item.codigo_in) && !naoEncontradosSet.has(item.codigo_in)`
         b. If disponiveis empty: return `{ fardos: [], quantidade_coberta: 0, quantidade_transformacao: demanda }` (Priority 4)
         c. Sort by `Math.abs(item.quantidade - demanda)` ascending
         d. If `sorted[0].quantidade >= demanda`: return `{ fardos: [sorted[0]], quantidade_coberta: demanda, quantidade_transformacao: 0 }` (Priority 1/2)
         e. Else: greedy descending sort by quantidade, take bales until `coberta >= demanda` or exhausted. `remainder = Math.max(0, demanda - coberta)`. Return `{ fardos: selected, quantidade_coberta: Math.min(coberta, demanda), quantidade_transformacao: remainder }` (Priority 3)
