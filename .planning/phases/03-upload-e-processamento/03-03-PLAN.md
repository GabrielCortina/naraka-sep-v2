1. Criar src/features/upload/hooks/use-upload.ts — hook 'use client' que gerencia todo o estado:

   **Estado:**
   ```typescript
   type UploadStep = 'idle' | 'file-selected' | 'parsing' | 'preview' | 'confirming' | 'success'

   const [step, setStep] = useState<UploadStep>('idle')
   const [file, setFile] = useState<File | null>(null)
   const [parseResult, setParseResult] = useState<ParseResult | null>(null)
   const [summary, setSummary] = useState<ImportSummary | null>(null)
   const [importacao_numero, setImportacaoNumero] = useState<number | null>(null)
   const [dayReset, setDayReset] = useState(false)
   const [error, setError] = useState<string | null>(null)
   const [imports, setImports] = useState<ImportRecord[]>([])
   const inputRef = useRef<HTMLInputElement>(null)
   ```

   **Funcoes:**
   - `handleDrop(e: React.DragEvent)`: preventDefault, pegar files[0], validar .xlsx (extensao), setFile, setStep('file-selected')
   - `handleFileSelect(e: React.ChangeEvent<HTMLInputElement>)`: pegar files[0], setFile, setStep('file-selected')
   - `handleDragOver(e: React.DragEvent)`: preventDefault (necessario para drop funcionar)
   - `handleProcess()`: setStep('parsing'), ler file como ArrayBuffer via FileReader, chamar parseXlsx(buffer), setParseResult, setStep('preview'). Tratar erros com setError e toast
   - `handleConfirm()`: setStep('confirming'), fetch POST /api/upload com body { rows: parseResult.rows, filtered_status, filtered_envio }, processar resposta, se sucesso: setSummary, setImportacaoNumero, adicionar ao imports[], toast sucesso 'Importacao #N concluida — X pedidos importados', se dayReset: toast info 'Virada de dia — pedidos anteriores removidos' ANTES do sucesso. setStep('success'). Setar timeout 3s para voltar a 'idle'
   - `handleUndo()`: window.confirm() com texto da UI-SPEC, fetch DELETE /api/upload/undo, remover ultimo import do imports[], toast 'Importacao #N desfeita'
   - `handleReset()`: voltar para step 'idle', limpar file, parseResult, summary, error
   - `fetchImports()`: Buscar importacoes do dia do Supabase via fetch GET (ou query client-side). Para simplificar: manter imports[] em estado local (carregado na pagina via server component e passado como prop)

   **Retorno do hook:**
   ```typescript
   return {
     step, file, parseResult, summary, importacao_numero, dayReset, error, imports,
     inputRef,
     handleDrop, handleDragOver, handleFileSelect, handleProcess, handleConfirm, handleUndo, handleReset,
   }
   ```
