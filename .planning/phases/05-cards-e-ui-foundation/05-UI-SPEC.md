## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, progress bar height |
| sm | 8px | Card internal element spacing, badge padding |
| md | 16px | Card padding, column gaps, modal padding |
| lg | 24px | Section padding, column header spacing |
| xl | 32px | Layout gaps between kanban columns |
| 2xl | 48px | Page-level vertical margins |
| 3xl | 64px | Not used this phase |

Exceptions:
- Touch targets on mobile: minimum 44px height for numpad buttons and action buttons (UIUX-06, D-25)
- Kanban column width: 220-260px fixed (D-02) -- not a spacing token, a layout dimension
- Progress bar height: 4px (D-10)

## Typography

| Role | Size | Weight | Line Height | Source |
|------|------|--------|-------------|--------|
| Body | 14px (0.875rem) | 400 (Regular) | 1.5 | Default |
| Badge/Label | 12px (0.75rem) | 700 (Bold) | 1.4 | D-50, D-51 |
| Countdown | 14px (0.875rem) | 700 (Bold) | 1.2 | D-52 |
| Pieces count | 12px (0.75rem) | 400 (Regular) | 1.4 | D-53, muted color |

Font weight budget: 2 weights for this phase (400, 700).

Additional rules:
- Badge metodo de envio: uppercase, 700, 0.75rem (D-50)
- Numero importacao (#1, #2): 700, 0.75rem, text-muted-foreground (D-51)
- Contagem regressiva: 700, color matches urgency tier (D-52)
- Pecas "300/900 pecas": 400, 0.75rem, text-muted-foreground (D-53)

Note: D-46 originally declared 3 weights (400, 600, 700). Semibold (600) dropped to comply with the 2-weight maximum. All roles previously using 600 now use 700 (Bold). Visual distinction is maintained through color and size rather than a third weight.

## Layout Contract

### Desktop (>=768px) — D-01 to D-07

```
+--AppShell (sidebar left)--+--Content Area---------------------------+
|                            | [Kanban Board - horizontal scroll]       |
|  Sidebar                   | [Col: SPX] [Col: Flex] [Col: Coleta]... |
|                            | [card]     [card]      [card]           |
|                            | [card]     [card]                       |
|                            |                                          |
|                            | [--- CONCLUIDOS (collapsed) -----------] |
+----------------------------+------------------------------------------+
```

- Columns: fixed width 220-260px each (D-02)
- Column order: by deadline ascending -- SPX(11h), Flex(12h), Coleta(14-16h), TikTok(15-18h), Shein(16h), Xpress(19h) (D-04)
- Empty columns hidden (D-05)
- Horizontal scroll if columns exceed viewport (D-02)
- Background: #f4f4f5 (D-06)

### Mobile (<768px) — D-12 to D-15

```
+--Content Area (full width)--+
| [Mobile Header]              |
| [v Shopee SPX - expanded]    |
|   [card full-width]          |
|   [card full-width]          |
| [> ML Flex - collapsed]      |
| [> ML Coleta - collapsed]    |
| [--- CONCLUIDOS (closed) --] |
| [Bottom Tabs]                |
+------------------------------+
```

- Columns become collapsible vertical sections (D-12)
- Cards full-width, 1 per row (D-13)
- Auto-expand: sections with overdue/urgent cards expanded, others collapsed (D-14)
- Tap card opens ItemModal (D-15)

## Interaction Contract

### Card Tap/Click

| Context | Action | Result |
|---------|--------|--------|
| Any device | Tap/click on OrderCard | Opens ItemModal (D-15, UIUX-05) |

### ItemModal Interactions (D-16 to D-24)

| Element | Action | Result |
|---------|--------|--------|
| Item row "Confirmar" button | Tap | Opens NumpadPopup with quantity reference |
| Item row "Nao Tem" button | Tap | Triggers cascade (busca fardo alt ou Transformacao) |
| Blocked item (AGUARDAR FARDISTA) | Visual | Badge shown, buttons disabled (D-21) |
| Print button (modal footer) | Tap | Generates PDF checklist (D-32, D-33) |
| Close modal | Tap X or overlay | Closes modal |

### NumpadPopup (D-25)

| Element | Action | Result |
|---------|--------|--------|
| Digit 0-9 | Tap | Appends digit to input |
| Backspace | Tap | Removes last digit |
| Confirmar (green) | Tap | Confirms quantity; if qty == needed: complete; if qty < needed: partial auto (D-20) |

Display: shows "Quantidade necessaria: X" as reference.

### Atribuicao (D-26 to D-30)

| Element | Action | Result |
|---------|--------|--------|
| Atribuir icon (card header) | Tap | Opens AssignModal with user list |
| Select user in AssignModal | Tap | Assigns user to card, icon turns blue + name |
| Role filter | Automatic | Shows separadores for prateleira cards, fardistas for fardo cards (D-26) |

### Urgency Transitions (D-36, D-37)

| Transition | Animation |
|------------|-----------|
| Green to Yellow (<2h) | Instant color change + subtle pulse on border (150ms) |
| Yellow to Red (0h) | Instant color change + pulse; countdown replaced by "ATRASADO" bold |
| Card completed (100%) | Fade out from column (300ms) + fade in at CONCLUIDOS section (200ms) (D-07) |
| Overdue card | Moves to top of column automatically (D-36) |

### Realtime Updates (D-34)

| Event | Visual Update |
|-------|---------------|
| Progress change | Progress bar fill updates, percentage updates |
| Card completed | Card animates to CONCLUIDOS |
| Assignment change | Atribuir icon updates |
| Urgency tier change | Border color + countdown text update |

```css
:root {
  --shopee: 14 89% 55%;       /* #ee4d2d */
  --ml: 54 100% 50%;          /* #ffe600 */
  --tiktok: 178 89% 55%;      /* #25F4EE */
```

```typescript
shopee: "hsl(var(--shopee))",

shein: "hsl(var(--shein))",
```
